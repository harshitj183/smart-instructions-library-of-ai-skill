# 🤝 Contribution Guide

We appreciate your interest in making the Smart AI Skills Library better! We are constantly fusing new technical breakthroughs from across the globe into our Mega-Skills and Master Roles.

## 🌟 How Can I Contribute?

### 1. **Submit New Skills**
Found a great set of instructions for a new framework (e.g., Qwik, Hono, SolidJS)?
- Create a new skill file in `skills/`.
- Compressed instructions are preferred over long paragraphs.
- High-fidelity logic blocks are key.

### 2. **Refine Existing Skills**
If you find a skill that causes hallucinations or is outdated (e.g., older React APIs):
- Open a PR to update the instructions.
- Add specific "Do's" and "Don'ts" to the file.

### 3. **AI Roles**
If you have a specialized master persona (e.g., Data Architect, Kubernetes Expert):
- Add a new master role to `roles/`.

## 🛠️ Requirements
- Files must be in Markdown (`.md`).
- Must follow the established compressed-instruction format.
- Logic should be strictly typed and actionable.

## 🚀 Getting Started
1. **Fork** the repository.
2. **Create** your feature branch.
3. **Commit** your changes.
4. **Push** to the branch.
5. **Open** a Pull Request.
